COMP 2156 DevOps Course

	COMP 2156 is a DevOps course that focuses on modern software development and system administration practices.
 The course introduces students to tools and techniques that improve collaboration between development and operations teams. DevOps helps organizations deliver software faster, more reliably, and with better quality.
	
	One of the main topics in COMP 2156 is version control using Git and GitHub. Students learn how to create repositories, manage branches, track code changes, and collaborate with team members using pull requests.
 Version control is important because it helps teams manage projects efficiently and prevents data loss.
The course also introduces Continuous Integration (CI) and Continuous Deployment (CD).
 Students learn how to automate testing and deployment using tools such as GitHub Actions and other CI platforms. Automation reduces human error and improves software reliability.
